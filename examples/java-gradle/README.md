# java-gradle

A Java based example with Gradle as its build system, inlined in the repository as a wrapper.

## Parts

* [api](api)
* [order](order)
* [token](token)

## Setup

`./gradlew` for OSX and Linux
`gradlew.bat` for Windows

### Manual

`./gradlew build`

### Intellij

`./gradlew idea`
Or open up `build.gradle` as a project in IntelliJ

### Eclipse

`./gradlew eclipse`
